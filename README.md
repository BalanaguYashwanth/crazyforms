# FormPilot (aka crazyforms)

    - Powered with MultiChain + Escrow + Rewards + LLM

# Tech stack

    - Frontend
        - Yarn, Scss, Js, REACTJS
    - Backend
        - Nestjs, Postgresql
    - Chains
        - Eth, KiiChain, SUI
